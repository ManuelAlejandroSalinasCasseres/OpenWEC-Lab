# Wave Measurement PCB

This board is responsible for the acquisition of hydrodynamic variables during laboratory-scale wave energy converter experiments.

## Main Components

- ESP32 Development Board
- HC-SR04 Ultrasonic Sensors
- Power Supply Circuit

## Repository Contents

- Schematics
- PCB Layout
- Gerber Files
- Bill of Materials (BOM)
- Images
- Firmware


This PCB was developed for laboratory-scale wave energy converter experiments, enabling synchronized acquisition of wave elevation data through an array of ultrasonic sensors.
